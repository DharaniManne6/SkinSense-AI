# SkinSense AI ✨

SkinSense AI is a **web application** that recommends skincare ingredients based on your **skin type** and **primary concerns**.  
It combines **AI-driven recommendations** with **interactive and cute animations** to provide a personalized and visually engaging user experience.


## Features
- Personalized ingredient recommendations based on **skin type** and **primary concern**  
- Shows “Good For” and “Avoid If” information for each ingredient  
- **Interactive UI** with pastel background and floating animations  
- **Responsive design** for mobile and desktop  
- Clickable **Learn More buttons** linking to external resources

## Technologies Used
- **Python & Flask** – Backend server and routing  
- **Pandas** – Data handling and filtering  
- **HTML, CSS, JavaScript** – Frontend structure, styling, and interactivity  
- **Lottie Web** – Animated vector graphics for skincare animations  
- **Custom JS / CSS** – Floating pastel blobs and mouse-following elements  
- **CSV** – Ingredient dataset (`ingredientsList.csv`)  

---

## AI Component
- Rule-based recommendation system matches skin type and concern to ingredients  
- Future enhancements can include:
  - **Computer Vision** – Skin type detection from selfies  
  - **NLP** – Analyze ingredient descriptions or reviews  
  - **Machine Learning Recommender Systems** – Personalized ingredient suggestions
